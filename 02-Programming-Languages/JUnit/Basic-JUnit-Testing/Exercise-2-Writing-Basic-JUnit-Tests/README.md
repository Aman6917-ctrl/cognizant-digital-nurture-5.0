# Exercise 2: Writing Basic JUnit Tests

## Objective

Write and execute basic unit tests using JUnit.

## Project Description

This exercise demonstrates unit testing of a Calculator class using JUnit 4.

The Calculator supports:

- Addition
- Subtraction
- Multiplication

JUnit test cases are used to verify the expected results.
