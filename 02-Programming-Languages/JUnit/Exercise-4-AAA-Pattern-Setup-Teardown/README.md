# Exercise 4: AAA Pattern, Setup and Teardown in JUnit

## Scenario

Organize unit tests using the Arrange-Act-Assert (AAA) pattern and use setup and teardown methods.

## Concepts Used

- Arrange-Act-Assert (AAA) Pattern
- @Before Annotation
- @After Annotation
- JUnit Assertions
- Maven

## AAA Pattern

### Arrange
Prepare test data and expected results.

### Act
Execute the method being tested.

### Assert
Compare the expected result with the actual result.

## Setup and Teardown

- `@Before` executes before every test method.
- `@After` executes after every test method.
