# Exercise 2: Test Suites and Categories

## Scenario

This exercise demonstrates how to group multiple JUnit tests into a test suite.

## Implementation

- Created a Calculator class.
- Created AdditionTest for testing addition.
- Created SubtractionTest for testing subtraction.
- Created AllTests test suite.
- Used @Suite annotation.
- Used @SelectClasses annotation to group multiple test classes.

## Technologies Used

- Java
- JUnit 5
- Maven

## Run Tests

```bash
mvn clean test
