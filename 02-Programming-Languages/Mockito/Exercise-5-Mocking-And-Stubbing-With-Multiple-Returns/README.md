# Exercise 5: Mocking and Stubbing with Multiple Returns

## Scenario

This exercise demonstrates consecutive return values from a mocked API.

## Implementation

- Created an ExternalApi interface and MyService class.
- Stubbed getData to return different values on consecutive calls.
- Verified both return values using JUnit assertions.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven

## Learning Outcomes

- Learned how to configure multiple return values with thenReturn.
- Tested consecutive calls to a mocked dependency.
