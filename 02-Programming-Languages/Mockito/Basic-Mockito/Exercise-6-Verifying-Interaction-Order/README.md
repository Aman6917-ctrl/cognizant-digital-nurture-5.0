# Exercise 6: Verifying Interaction Order

## Scenario

This exercise demonstrates verifying the order of method interactions.

## Implementation

- Created an ExternalApi interface with connect and fetchData methods.
- Created a service that connects before fetching data.
- Verified the call sequence using Mockito InOrder.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven

## Learning Outcomes

- Learned how to verify interaction order with InOrder.
- Confirmed that dependent operations occur in the required sequence.
