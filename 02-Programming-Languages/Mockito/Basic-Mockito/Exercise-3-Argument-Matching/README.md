# Exercise 3: Argument Matching

## Scenario

This exercise demonstrates verifying a method call with an argument matcher.

## Implementation

- Created an ExternalApi interface with sendData.
- Created a MyService class that sends data through ExternalApi.
- Verified the supplied argument using the eq matcher.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven

## Learning Outcomes

- Learned how to use Mockito argument matchers.
- Verified that a method received the expected argument.
