# Exercise 4: Handling Void Methods

## Scenario

This exercise demonstrates stubbing and verifying a void method using Mockito.

## Implementation

- Created an ExternalApi interface with a void sendData method.
- Stubbed the method with doNothing.
- Verified the interaction after calling the service.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven

## Learning Outcomes

- Learned how to stub void methods using doNothing.
- Verified a void method interaction.
