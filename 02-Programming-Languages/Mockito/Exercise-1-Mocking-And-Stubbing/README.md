# Exercise 1: Mocking and Stubbing

## Scenario

This exercise demonstrates mocking and stubbing using Mockito.

## Implementation

- Created an ExternalApi interface.
- Created a MyService class that depends on ExternalApi.
- Created a mock object using Mockito.
- Stubbed the getData method using when and thenReturn.
- Verified the returned value using assertEquals.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven
