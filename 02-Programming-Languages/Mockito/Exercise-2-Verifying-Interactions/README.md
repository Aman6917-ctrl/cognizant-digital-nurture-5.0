# Exercise 2: Verifying Interactions

## Scenario

This exercise demonstrates verifying an interaction using Mockito.

## Implementation

- Created an ExternalApi interface.
- Created a MyService class that depends on ExternalApi.
- Created a mock object using Mockito.
- Called the fetchData method.
- Verified that getData was called using verify.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven

## Learning Outcomes

- Learned how to verify method interactions using Mockito.
- Confirmed that a dependency method was called by a service.
