# Exercise 2: Mocking External Services

## Scenario

This exercise demonstrates testing a service that calls an external RESTful API.

## Implementation

- Created a RestClient interface.
- Created an ApiService class that depends on RestClient.
- Created and stubbed a mock REST client.
- Verified the returned value and REST client interaction.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven
