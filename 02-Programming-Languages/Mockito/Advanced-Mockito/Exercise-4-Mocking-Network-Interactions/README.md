# Exercise 4: Mocking Network Interactions

## Scenario

This exercise demonstrates testing a service that interacts with a network resource.

## Implementation

- Created a NetworkClient interface.
- Created a NetworkService class that depends on NetworkClient.
- Created and stubbed a mock network client.
- Verified the returned value and network client interaction.

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven
