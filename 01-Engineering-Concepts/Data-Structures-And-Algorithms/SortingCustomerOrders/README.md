# Sorting Customer Orders

## Objective

Implement Bubble Sort and Quick Sort algorithms to sort customer orders based on their total price.

---

## What are Sorting Algorithms?

Sorting algorithms arrange data in a specific order. In this project, Bubble Sort and Quick Sort are used to sort customer orders by total price.
