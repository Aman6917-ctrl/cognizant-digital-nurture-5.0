# Employee Management System

## Objective

Implement an Employee Management System using Arrays in Java to perform basic operations such as add, search, traverse, and delete employee records.

---

## What is an Array?

An Array is a linear data structure that stores elements in contiguous memory locations. It provides fast index-based access but has a fixed size.
