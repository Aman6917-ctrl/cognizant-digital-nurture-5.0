# E-commerce Platform Search Function

## Objective

Implement Linear Search and Binary Search algorithms in Java to search products efficiently.

---

## Problem Statement

Develop a product search system for an e-commerce platform using Linear Search and Binary Search.

---

## Algorithms Used

- Linear Search
- Binary Search

---

## Project Structure

```text
EcommerceSearchSystem
├── README.md
└── src
    ├── Product.java
    ├── Search.java
    └── Main.java
```

---

## How to Compile

```bash
cd src
javac *.java
```

---

## How to Run

```bash
java Main
```

---

## Expected Output

```text
Linear Search Result:
ID: 103, Name: Keyboard, Category: Accessories

Binary Search Result:
ID: 104, Name: Monitor, Category: Electronics
```

---

## Time Complexity

| Algorithm | Best | Average | Worst |
|------------|------|----------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |
