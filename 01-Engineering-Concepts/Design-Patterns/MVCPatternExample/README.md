# MVC Pattern Example

## Objective

Implement the MVC (Model View Controller) Design Pattern in Java to separate data, business logic, and presentation.

---

## What is MVC Pattern?

MVC divides an application into three parts:

- Model → Stores data
- View → Displays data
- Controller → Connects Model and View
