# Singleton Pattern Example

## Objective

Implement the Singleton Design Pattern in Java to ensure only one instance of the Logger class is created.

## What is Singleton Pattern?

Singleton is a Creational Design Pattern that ensures only one object of a class exists throughout the application and provides a global access point to it.
